Tugas_MVC_PBKK_B

Tugas ini saya membuat aplikasi web untuk membuat sebuah Note atau catatan berbasis web, sehingga kita dapat menyimpan catatan kita dalam web.

ada 3 halaman/page yang tersedia dalam web ini.

1. Index (Home page)
2. addNotes (Input)
3. noteLists (Output)

halaman pertama yang ditampilkan adalah halaman home page dari web, lalu akan ada 2 tombol yang tersedia di halaman tersebut yang akan menyambungkan dengan 2 halaman lainnya.

tombol pertama menyambungkan user dengan halaman Input dimana user dapat memasukkan inputan yang nanti akan ditampilkan di halaman outputnya.

tombol kedua adalah tobo untuk melihat halaman output dimana user sapat melihat hasil output dari input yang telah dimasukkan.
